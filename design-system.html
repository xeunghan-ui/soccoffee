00

// 현재 진단

싸커피는 3개의 정적 HTML(소개·멤버앱·팀빌더)이 각자 인라인 <style>을 들고 있고, 공유 토큰 파일이 없다. 색은 일부 변수화돼 있으나 페이지마다 체계가 다르다.

왜 마음에 안 드나 — 구조적 원인 5가지

1. 토큰 체계가 분리 — 소개는 --green/--sand/--cream, 멤버앱은 --bg/--card/--coffee/--accent. 같은 hex를 다른 이름으로 중복 정의(#15281b가 --green이자 --bg, #ddd6b6이 --sand이자 --accent이자 --green(멤버)).
2. 오명명 토큰 — 멤버앱 --green과 --red가 둘 다 샌드(#ddd6b6)를 가리킨다. 의미 색이 없다.
3. 디스플레이 폰트 불일치 — 소개는 Anton, 멤버앱은 WildGag(로드 불명확). 큰 글자 인상이 페이지마다 다르다.
4. 평평한 위계 — 굵기가 700·800·900에 몰려(합 86곳) 다 굵다 → 강조가 강조 같지 않다. 본문도 무겁다.
5. 에너지 컬러 부재 — 전체가 그린·샌드·크림뿐. 차분을 넘어 밋밋하고, CTA·활성·D-day가 눈에 안 띈다.
토큰 부재의 결과 — 폰트 크기 25종+(하프픽셀 10.5px 포함), radius 15종, gap/padding 제각각. 전부 96개 인라인에 하드코딩. 이 스펙은 그걸 수렴할 단일 캐논을 정의한다(색은 통합·정명명, 타입/간격/radius는 신규 토큰화).

02

// 타이포 (Type)

디스플레이는 Anton 하나로 통일(콘덴스드 대문자 = 저지 넘버·클래식 스포츠). 본문은 Pretendard. 크기는 25종+를 정수 ramp로 수렴, 하프픽셀 폐기.

디스플레이 — Anton (콘덴스드)

--fd-156MATCH DAY히어로
--fd-24007.01 WED날짜·섹션
--fd-32820 / 30 납부수치·통계
날짜·등번호·D-day·스코어·큰 수치 = Anton. 대문자 + 좁은 자간으로 클래식 키트 느낌. WildGag는 로고 워드마크에만 남기고 디스플레이 면은 Anton으로 단일화.

본문 — Pretendard Ramp

--fs-2xl22이번 주 매치← 22/23/24
--fs-xl18섹션 제목← 17/18/19/20
--fs-lg16카드 제목·강조 본문← 16
--fs-base14기본 본문 — 참석/회비 안내 문구← 13/13.5/14/14.5/15
--fs-sm13보조 본문·리스트 텍스트← 12.5/13
--fs-xs12라벨·메타·뱃지← 11.5/12
--fs-micro11최소 태그 — 가독성 하한← 7/8/9/10/10.5
하프픽셀·초소형 폐기. 10.5px 등은 의도가 아니라 드리프트, 기기별 서브픽셀 반올림으로 렌더가 들쭉날쭉. 7–10px 텍스트는 딥그린 위에서 거의 안 읽힘 → micro 11px 하한으로 흡수(취향 아닌 가독성 결정). 본문 기준은 13→14로 올려 한 호흡 키운다.

굵기 — Weight

--fw-regular 400본문 기본 — 평소 텍스트는 여기
--fw-medium 500약한 강조·활성 라벨
--fw-bold 700라벨·소제목·강조
--fw-extra 800히어로·디스플레이 보조
위계 복구. 현재 700·800·900이 86곳 — 다 굵으니 강조가 안 산다. 본문을 400/500으로 내리고 700은 진짜 라벨/제목에, 800은 최상위에만. 900(20곳)은 폐기 → 800, 600은 700에 흡수. 굵기 5→4단.

03

// 모서리 (Radius)

현재 15종. 5단으로 수렴. 레트로 스포츠는 과한 라운딩보다 약간 또렷한 모서리가 어울려 카드 기준을 12px로 못박는다(현재 --radius 14는 12로).

--r-sm6px
칩·인풋 ←2~7
--r-md10px
버튼·작은 카드 ←9~11
--r-lg12px
카드 기본 ←12/14/20
--r-pill999px
뱃지·탭
--r-circle50%
아바타·도트
12px(120회)·10px(67회)·999px(60회) 우세 그대로 채택. 20px·14px → 12px lg로, 2~8px 소형 → 6px sm으로 흡수.

04

// 간격 (Spacing)

4px 그리드. gap 최다인 6px·8px을 살린 8단 스케일. gap·margin 우선 적용, 비대칭 padding은 케이스별로.

--sp-1 4
--sp-1h 6 gap 2위(115회)
--sp-2 8 gap 1위
--sp-3 12
--sp-4 16
--sp-5 20
--sp-6 24
--sp-8 32
현재 우세 padding 8px 4px·9px 14px·0 16px 등 비대칭이 많아 단일 토큰화는 한계 → 스케일은 gap·margin·세로 리듬에 먼저 적용.

05

// 컴포넌트 방향

새 토큰이 실제로 어떻게 보이는가 — 레트로 골드 + 의미 색이 들어간 핵심 위젯 미리보기.

탭 — 활성=골드
회비참석랭킹
버튼
상태 뱃지 — win / alert / gold
납부미납D-2
디스플레이 수치 — Anton
20/ 30 납부 · 67%
레트로 시그니처 — ① 디스플레이/수치/등번호는 Anton 대문자, ② 라벨·섹션 헤더 대문자+자간, ③ 활성·CTA·D-day는 골드, ④ 카드 상단/섹션에 골드 1px 라인, ⑤ 배경에 은은한 피치 라인(이 문서 배경 참고). 차분한 그린 위에 골드가 포인트로 튀며 클래식 스포츠 클럽 인상을 만든다.
