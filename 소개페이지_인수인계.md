# 소개 페이지 인수인계 (index.html)

싸커피(SOCCOFFEE FC) **공개 랜딩/소개 페이지**. 전체 배경은 `인수인계.md`, 작업 규칙은 `CLAUDE.md` 참고.

## 기본
- **파일**: 루트 `index.html` (⚠️ 두 개 있음 — 루트=소개, `team/index.html`=팀빌더. **루트를 팀빌더로 덮지 말 것.** 파일 첫 줄 주석으로 식별).
- **URL**: `https://xeunghan-ui.github.io/soccoffee/`
- **성격**: 로그인 불필요, 공개. 히어로의 "멤버 로그인 →" 버튼이 `member.html`로 연결.
- 정적 HTML 단일 파일(인라인 `<script>` 1개) + Supabase 읽기 전용.

## 구성(섹션 순서)
1. **히어로** — SCF 로고(`scf_logo.png`) + "SOCCOFFEE" 워드마크 + CTA(멤버 로그인 / 인스타).
2. **매치데이 바 / 스탯** — 픽스처 요약, 스코어보드형 통계.
3. **OUR KITS** — WHITE / BLACK 키트.
4. **MATCH DAY** — 경기 일정. **Supabase `club_settings(id='current').sessions`** 자동 표시. 세션의 `placeUrl` 있으면 장소가 링크(↗). `desc` 있으면 행 클릭 시 펼침. 오틀리 행사 줄은 `/oatly`로 이동.
5. **SQUAD** — **`club_settings(id='teambuilder').players`** 명단(former·friends 제외, 등번호순). **전 달(확정) MVP·성장캐 배지** 표시(`potm_votes`에서 직전 calendar month 집계 — 진행 중인 달은 표시 안 함).
6. **CAFES / MOMENTS(갤러리)**.

## 데이터 (모두 읽기 전용, anon 키)
- `club_settings` `id='current'`(세션), `id='teambuilder'`(명단) / `potm_votes`(전 달 MVP·성장).
- 세션·명단은 **멤버앱·팀빌더에서 관리** → 소개는 새로고침 시 자동 반영. 소개에서 데이터를 쓰지 않음.

## 디자인 토큰(멤버 페이지와 통일, `design-system.html` 캐논)
- 색: `--green#15281b`(배경) `--green-2#1e3526` `--sand#ddd6b6` `--cream#ece6d2` `--muted#a4b39a` `--line#33492f`. 골드는 샌드로 흡수(별칭만 유지). 상태 `--win#5a9277` `--alert#bd6452`.
- 디스플레이 폰트 **`--disp:"Oswald"`** (Anton에서 교체 — 더 가벼움). 굵기 3단(400/600/800), 라벨·헤딩 600.
- OG 이미지: `img/gal-ssau-1.jpg`. 홈 추가 아이콘: `scf-icon-180.png`.

## 자주 하는 작업
- **갤러리 사진**: `img/`에 넣고 **PIL로 압축**(1920px·q82·`exif_transpose`) 후 `.gallery`에 `<img loading="lazy">` 추가.
- **OG/대표 이미지**: `index.html` head의 `og:image`/`twitter:image` 수정.
- **MVP·성장 배지가 안 뜸**: 전 달 `potm_votes`가 없으면 정상(미표시). 이번 달 진행 중인 투표는 일부러 표시 안 함.
