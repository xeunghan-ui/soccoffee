## 주요 기능
- **선수 ⚙️ 관리**: 🏅티어 변경 / 🔤영문명 / 🔢등번호 / 📆**가입일 설정** / 📅휴면 월 관리 / 💤휴면 전환 / 🚪탈퇴. (삭제는 없음 — 탈퇴 처리만)
- **회원 추가 폼**: 이름·성별·티어·등번호 + **가입일(기본 오늘)**. ⚠️ 가입일 없으면 사이트 참석·회비에서 제외되니 꼭 설정.
- **다음달 휴면 자동 롤오버**: 매월 15일 이후 팀빌더를 열면, 이번 달 휴면(또는 영구휴면) 회원의 **다음 달을 dormantMonths에 1회 자동 추가**(`dormRollover` 마커로 월 1회). 멤버가 멤버앱에서 '활동'으로 바꾼 건 덮어쓰지 않음. (멤버앱도 로드 시 같은 롤오버 1회 수행)
- **2팀 자동 배분**, 출석 기록, "🔄 사이트 출석"(사이트 참석 yes를 빌더로 가져옴), 회비/용품/랭킹 등.
- 선수 관리 팝업은 화면 밖으로 안 잘리게 위치 보정됨.

## ⚠️ 핵심 주의 — 신규 회원 id
- 팀빌더 `addPlayer`는 **`id = Date.now()`**(매우 큰 수)를 부여. 기존 SEED 멤버는 1~39.
- 이 큰 id가 들어가는 DB 컬럼은 **반드시 `bigint`** 여야 저장됨: `dues.member_id`, `attendance.member_id`, `potm_votes.voter_id/candidate_id`. (`int`면 신규 회원의 회비·참석·투표가 저장 실패) — `Supabase_전체설정.md`에 ALTER 포함.

## 휴면 규칙
- 휴면은 **월 단위**(예: '2026-07')로 `dormantMonths`에 기록. 사이트의 회비·참석·명단에서 해당 월 자동 제외.
- 멤버앱 홈 상태창에서 멤버 본인이 **다음 달 활동/휴면을 셀프 토글**(2026-07-15부터) → 팀빌더 dormantMonths에 직접 반영.
